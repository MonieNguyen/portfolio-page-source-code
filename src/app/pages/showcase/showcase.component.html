Node-RED Project

Project Overview
Adward Best project of the course
Background This is a project under the subject: "Could Computing" in my master's course.
Project Objectives Use Node-RED and various cloud services to develop an Enterprise Resource Planning Application module
Team Size 5 people Duration 8 weeks (Effort: ~10h / week / person)
Scope of work

Develop a grocery stores management system consiting of three main components

  1. Node-RED Editor allowing administrators to modify the system's source code
  2. Admin dashboard allowing administrators to view store sales, earnings, and analytics insights
  3. Client website allowing store users to view or search product, interact with a chatbot
System main features Store Administrator User:
  1. Login
  2. Add/Edit/Delete source code of the whole system
  3. View admin dashboard
  4. Receive notification email about products nearly out-of-stock or out-of-date
Store Customer User:
  1. View store information
  2. View and search product
  3. Interact with a chatbot
