Project Overview
Background This is a project under the subject: "Cloud Computing" in my master's course.
Adward Best project of the course Grade High Distinction
Project Objectives Use Node-RED and various cloud services to develop an Enterprise Resource Planning Application module
Team Size 5 people Duration 8 weeks (Effort: ~10h / week / person)
Scope of work

Develop a grocery stores management system consiting of three main components

  1. Node-RED Editor allowing administrators to modify the system's source code
  2. Admin dashboard allowing administrators to view store sales, earnings, and analytics insights
  3. Client website allowing store users to view or search product, interact with a chatbot
System main features Store Administrator User:
  1. Login
  2. Add/Edit/Delete source code of the whole system
  3. View admin dashboard
  4. Receive notification email about products nearly out-of-stock or out-of-date
Store Customer User:
  1. View store information
  2. View and search product
  3. Interact with a chatbot
Technologies and Services Used
  • IBM Cloud Services: IBM Cloud Foundry, IBM Cloud Object Storage, IBM Watson Assistant
  • Node-RED (flow-based programming tool) and its palette
  • Angular
  • Nectar cloud instance

Data Analysis Project

Grade High Distinction
Project Requirements Collect data on a specific topic from the Twitter API or Reddit and perform some analysis on the collected data.
Team Size 3 people Duration 6 weeks (Effort: ~10h / week / person)
Our team solution
  1. Use R to collect ~28k tweets including the hashtag "#web" from Twitter API
  2. Clean data: Remove special characters, words less than 3 characters, hashtags, tagged accounts, English stop words, links, emoji
  3. Pre-process data: split into words, convert to required data type
  4. Perform 10 different types of data analysis ranging from simple to complex
Technologies Used R (a good programming language for data analysis and data visualisation)
